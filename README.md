# Music Genre Prediction

Este repositório contém scripts para a extração de características de áudio e geração de espectrogramas, com o objetivo de treinar modelos de machine learning para previsão de gênero musical. O principal objetivo é testar diferentes abordagens e técnicas para identificar qual delas oferece o melhor desempenho na classificação de gêneros musicais.

## Estrutura Principal
- **scripts/**: Scripts para pré-processamento de áudio, criação de espectrogramas, treino e predição dos modelos.
- **models/**: Modelos de machine learning treinados.
- **data/**: Dados de áudio utilizados no treinamento e teste.
- **results/**: Resultados das predições e avaliações dos modelos.
- **converted_wavs/**: Áudios convertidos no formato wav usados pelos scripts.

## Dependências
As dependências necessárias estão listadas em `requirements.txt`. Para instalar, utilize:

```bash
pip install -r requirements.txt
```
