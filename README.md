# Music Genre Prediction

Este repositório contém scripts para a extração de características de áudio e geração de espectrogramas, com o objetivo de apoiar projetos de classificação de gênero musical.

## Scripts

- `scripts/generate_csv_from_audio.py`: Extrai diversas características de áudio de arquivos `.wav` e as salva em dois arquivos CSV.
- `scripts/generate_spectograms.py`: Gera espectrogramas de Mel a partir de arquivos `.wav`, salvando-os como imagens PNG para faixas completas e segmentos.
